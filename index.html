How to set up a VS Code shortcut on Windows

This quick walkthrough will show you how to make an option in your right-click context menu that functions the same as our "code ." command in git bash. This will make your life much easier when navigating some complex folder structures like the class gitlab repository.

Warning: This requires that you make some simple changes to your Windows Registry. If you are uncomfortable with making these registry edits, please do not attempt to follow this guide. Changing the wrong fields of the registry can damage your installation of Windows. You have been warned.

Step 1: Locate your VS Code installation

If you're unsure of where your copy of VS Code is located, you can find it easily by right-clicking on your VS Code shortcut—on your desktop, taskbar, or start menu—and selecting "Open File Locaton".

If this brings you to another shortcut, simply repeat the process on the new shortcut until you reach your VS Code executable. Make note of this path. In my example, the path is D:\Program Files\Microsoft VS Code\Code.exe

Step 2: Open Registry Editor

To open the registry editor, hit ⊞ Win + R and it should open a run prompt as shown above. Type in "regedit" and hit OK to open the registry editor.

Step 3: Navigate to shell settings in registry

Inside your registry, navigate to Computer\HKEY_CLASSES_ROOT\Directory\Background\shell

(Be sure you are looking past the registry keys that start with ".")

Step 4: Create a new key for VS Code

Right-click on the shell folder and select New → Key. Name this key something logical, like "VS_Code". Then, right-click on your new key and create yet another key inside this one; call this child key "command"

Step 5: Edit the command string

In the command key, right-click the default string value where it says (Default) and select Modify...

In the Value data field, fill in the path to your VS Code executable from step 1, followed by a space and then "%V". In my example, the total string is D:\Program Files\Microsoft VS Code\Code.exe "%V"

Important: You should only wrap the %V in quotes. Any other quotes in this string may break its functionality.

Now your shiny new menu option should exist! Right-click inside of any folder and select the new option (labeled VS_Code by default) to open VS Code in that folder.

Extra Options: Make it more pretty!

If you want your context menu to have an icon or better text, then we need to add some data to the VS_Code key directly. To start, Modify... the default string to give the name of the menu option.

To add an icon to the context menu option, right-click and select New → String Value. Give this value the name "Icon", and set its data to be the path to your VS Code executable.
